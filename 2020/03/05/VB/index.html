一开始我以为，已经停更了二十几年的VB已经没有太大加载，在昨天我研究如何在office2016上如何运行flash的时候（flash已经快停止维护了，还用flash做PPT的老师可长点心吧），发现了一个神奇的功能：

中间的一些功能似曾相识，拖动几个看看

咦咦咦？这是要在PPT中跑程序吗？

双击任意控件，弹出编程窗口：

这就是VB没错了。更加准确地说，应该叫VBA，VB宏语言，可以替代人工地一些按键操作，并且在Office套件里面都是支持的。

这就意味着，使用PPT不仅仅只是页面切换，还可以有一种更加高级的玩法了。比如基于PPT的程序，自动整理数据等……如此方便的东西，还不赶快学起来！

首先我们来认识下VB的数据类型。

类型 标识符 简写
整数 Integer %

定义变量

Dim [变量名] As [数据类型]

或者使用简写

Dim [变量名][简写]

或者直接进行赋值

[变量名] = [值]

常量

定义常量

Const [常量名] = [数值]

数组

定义数组

Dim [数组名](数组个数) As [数据类型]

或者简写为

Dim [数组名](数组个数)

###　指定角标

Dim [数组名]([起始值] to [结尾值])

定义多维数组

定义个数A×数组个数B×…个数据

Dim [数组名](数组个数A, 数组个数B, ...) As [数组类型]

自定义类型

定义自定义类型

有时我们需要一些别的数据类型，我们可以进行自定义

Type [类型名]
<子类型> As <数据类型>
<子类型> As <数据类型>
...
End Type

赋值自定义类型

[变量名] As [类型名]
[变量名].[子类型] = [赋值]

同时可以使用 With 关键字

With [变量名]
.[子类型] = [赋值]
End With

函数

自定义函数

Private/Public Static Function [函数名]([参数], [参数], [...]) As [数据类型]
        [函数语句]
        [函数名] = k ‘相反于return返回值
End Function

使用函数

[对象] = [函数名](参数)

简单的，也可以

[函数名](参数)
