Frame框架

选项 说明
master 父组件 要将组件放置的位置

Label文本框

类型 说明
master
text 字符串 文本
bg 字符串 背景色
fg 字符串 前景色
font ([表示字体的字符串]， 大小) 字体
width 数字 宽度
height 数字 高度
justify “left”, “center”, “right” 文本对齐的方式

Entry单行文本框

选项

类型 说明
master
show 字符 作为密码时显示的内容
textvariable tkinter.Variable() 替代文本选项资源
width

方法

获取文本

Entry.get()

删除文本

Text.delete([整数索引1]，[整数索引2])
整数索引
0 开头
tkinter.INSERT 光标
tkinter.END 结尾

Text多行文本框

选项

类型 说明
master
width
height

方法

插入文本

Text.insert([浮点数索引]， 文本)
浮点数索引
0.0/1.0 在开头插入
tkinter.INSERT 在光标插入
tkinter.END/2.0 在结尾插入

获取文本

Text.get([浮点数索引1]，[浮点数索引2])

清除文本

Text.delete([浮点数索引1]，[浮点数索引2])

Scrollbar滚动条

绑定

在能够向下滚动的部件 如 多行文本框、列表等参数中加入yscrollcommand=scroll.set（同样也有xscrollcommand=scroll.set），即可绑定滚动条。

然后再把函数

控件Scrollbar.config(command=[绑定的控件].yview)

Listbox列表

选项

类型 说明
master
selectmode “single”、”browse”、”multiple”、”extended” 选择模式。单选，拖动/按动多选，多选，Ctrl/Shift多选

方法

删除

Listbox().delete([索引1]，[索引2])

Listbox().get([索引1]，[索引2])

Menu菜单

选项

选项 说明
master

方法

菜单条

menubar = tkinter.Menu(win)
master.config(menu=menubar)

菜单项

添加项头

选项 说明
master
tearoff 布尔类型 菜单可否单独显示 
menu = tkinter.Menu(menubar, tearoff=False)
menubar.add_cascade(label=[显示的字符串], menu=menu)

添加内容

menu.add_command(label=[显示的字符串], command=[函数])
