画图要下载matlab？但我看到19G的下载后，小内存的固态硬盘哭了。

不想下载matlab？没有关系，用Python + mathlabplot照样可以画三维图！

而现在只要11行代码就可以轻松完成函数画图！

很漂亮有没有？！

画曲面图：

import matplotlib.pyplot as plt
import numpy as np

ax = plt.axes(projection='3d')


def f(x, y):
    return (1- x**2 - y**2)**0.5    # 在这里写函数


x = np.arange(-1, 1, 0.1)   # 定义域
y = np.arange(-1, 1, 0.1)   # 精细程度
X, Y = np.meshgrid(x, y)
Z = f(X, Y)


ax.plot_surface(X, Y, Z, rstride=1, cstride=1, cmap='viridis', edgecolor='none')

plt.show()

画等高线图：

import matplotlib.pyplot as plt
from mpl_toolkits import mplot3d
import numpy as np


ax = plt.axes(projection='3d')


def f(x, y):
    return (1-x**2 - y**2)**0.5    # 在这里写函数


x = np.arange(-1, 1, 0.1)   # 定义域/精细程度
y = np.arange(-1, 1, 0.1)
X, Y = np.meshgrid(x, y)

Z = f(X, Y)

ax.contour3D(X, Y, Z, 50, cmap=plt.cm.jet)
plt.show()
