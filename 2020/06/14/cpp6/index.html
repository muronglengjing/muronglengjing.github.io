类与对象

定义类与对象

说明语句

  • class <类名>
    {
    public:
    //
    protected:
    //
    private:
    //
    }

    • private

      • 默认值
      • 类内可见
      • 不能在类外和派生类内使用

    • protected

      • 类和派生类内可见

    • public

      • 类和类外可见

  • struct也可以声明类

    • 默认所有的对象都是公有的

类外声明

返回类型 类名::函数名(参数表)

访问对象成员

与访问结构的形式相同

  • .
  • ->

this指针

指对象本身

常指针

构造函数

类名::类名(参数表);

建立类对象时自动调用

不能通过对象显式调用

如果数据成员含有指针

构造函数复制时仅复制指针的地址值

析构函数

类名::~类名();

类对象结束后自动调用

常成员函数

返回值 函数名() const

函数体内不能修改对象的成员

静态成员

在类内声明，在类外定义

  • 类型 类名::成员标识符= 初始值;

访问方式

  • 类名::成员标识符
  • 对象访问

静态成员函数

  • 类名::函数名
  • 不依赖类数据结构，没有this指针
  • 只能访问类静态成员

友元

可以访问类的所以成员 （含私有成员）

是非对称、非传递的

成分

  • 友元函数

    • 不受声明位置（public等）的影响

    • 不是类的成员，通过参数访问类的成员

      • 在参数表中必须显式地指明要访问的对象

  • 友元类

    • 友元类中所有的函数都是其基类的友元函数

参数初始式

构造函数名 (变元表):数据成员1(变元表),…,数据成员n(变元表){};

赋值形式不能调用类的类成员的构造函数，需要用参数初始式来进行调用
